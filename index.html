Projeto de Análise com Python

Baseado no código do Google Colab

Aula 1 — Introdução

Nesta aula aprendemos os conceitos básicos de Python e como utilizar o Google Colab.

Aula 2 — Manipulação de Dados

Exploramos bibliotecas como Pandas e Numpy para trabalhar com dados.

Aula 3 — Visualização de Dados

Usamos Matplotlib e Seaborn para criar gráficos informativos e bonitos.

Aula 4 — Modelagem

Aplicamos algoritmos de aprendizado de máquina usando Scikit-learn.

Aula 5 — Conclusões

Apresentamos as conclusões da análise e discutimos próximos passos.
